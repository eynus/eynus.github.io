axios

axios简介

axios(发音：阿k斯依欧z or 阿k修z or whatever:)

axios是一个基于Promise的HTTP库，可以用在浏览器和node.js中。

axios的特性：

  • 是一个基于Promise的HTTP库，支持 Promise API
  • 能够拦截请求和响应
  • 可以转换请求数据和响应数据 ，自动转换 JSON 数据
  • 安全性更高，客户端支持防御 XSRF
