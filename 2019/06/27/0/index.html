如何理解[]！==“0”

所以为什么"0" == []是false?

在进行==运算时，需要记住几条规则。(详见ES5标准-11.9.3)

解释 ->
1.[] == 0等价于 ToPrimitive([]) == 0等价于 [].valueOf().toString() == 0等价于''==0,对照第四条规则，将字符串转化为数字类型，得到0==0,因此为true
2.[] == '0'等价于 ToPrimitive([]) == '0'等价于等价于''=='0',等号两边类型相同，参考第一条中d规则，''!=='0',因此为false
(了解toPrimitive)
