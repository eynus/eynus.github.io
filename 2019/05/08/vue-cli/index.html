认识vue-cli

概念

vue-cli是vue.js的脚手架，使用它来快速创建项目，能够集成Babel、ESLint,css预处理器，等项目的各个配置。起到一个前端自动化构建工具的作用，承包了脏活累活。

安装步骤

1.安装vue-cli

要求node -v 版本大于8.9，如果小于8.9，需要卸载重装

2.检查安装是否成功

如果version<3.0需要卸载重装 （npm uninstall vue-cli -g）

3.使用.

  1. 进入某个文件夹
  2. vue create [项目名称(小写用-隔开)]

  1. 配置

    1. 工具选择(空格键操作)——> babel/linter——>Enter

    2. 代码规范——> 选择仅一个错误——>Enter

    3. linter特性——>默认——>Enter

    4. 独立的配置文件——>Enter

    5. 以上配置选择进行保存——>Y——>保存为一个名字

      ![](/img/55.png)

  2. 安装完毕，启动。

  1. cd [vue文件夹]

  2. npm run serve

  3. localhost:8080打开
