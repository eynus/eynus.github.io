h5-鼠标拖放drag&drop

Drap和Drop拖放操作

1.拖放元素

有以下事件：

事件 描述
dragstart 被拖放元素开始被拖拽时触发
drag 被拖放元素在拖拽过程中触发
dragend 被拖放元素拖拽完成时触发

2.被拖放元素

事件 描述
dragenter 拖放元素进入目标元素时(以鼠标位置是否进入为准)
dragover 拖放元素在目标元素上时
dragleave 拖放元素在目标元素上离开时
drop 拖放元素在目标元素上同时鼠标放开时触发(e.g 文件上传/拖动删除,需要同时取消dragover默认事件)

3.存储数据：DataTransfer对象

方法 描述
getData() 从DataTransfer对象读取数据
setData() 向DataTransfer对象存放数据
