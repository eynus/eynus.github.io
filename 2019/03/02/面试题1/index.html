如何判断一个js是否是数组

  • arr instanceof Array (但是如果是继承数组的 也是true的)
  • arr.constructor == Array
  • O.prototype.toString.call(xxx) ==”[object Array]” 这是 JQuery方法
  • isArray ES5之后的方法(不需要管它实在哪个全局环境创建。因为之前的一些方法都有弊端，比如1如果是在两个以上不同的执行环境两个的构造函数就不一样)
