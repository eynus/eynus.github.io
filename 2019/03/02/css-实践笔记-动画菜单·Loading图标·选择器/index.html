CSS动画效果菜单

  1. font-face

    它允许网页开发者为其网页指定字体。 通过这种作者自备字体的方式，@font-face 可以消除对用户电脑字体的依赖。

    字体库下载地址：https://www.dafont.com/

    找字网 http://www.zhaozi.cn/s/all/ttf/index_2.php

  2. 能不脱离文档流就不脱离

    能用inline-block就不要用float,poisition

  3. text-shadow 文字阴影

    ——使文字细腻 参数：x y blur color

    影响性能，不要使用过多

  4. box-shadow 盒子阴影

    x y blur spread color inset

  1. transition 过渡

    icon图标字体大小变化的过渡

  2. animation 动画
  3. transform 元素变换

    (旋转，倾斜，缩放)

    1. 利用transform使未知宽度高度的块元素水平垂直居中

    2. 动画 使用transform 或 (使用width,margin,top值实现)——transform优化性能

      • (使用width,margin,top值实现)每变化一次，主线程运行一次，辅线程是一样的变化n次
      • transform,主线程只需要变化一次，辅线程是一样的变化n次
  4. inline-block之后的设置

    将li设置为行级块元素后，可以视作普通行内元素，设置text-align

  5. 设置为行级块元素后，设置vertical-align:middle

  6. z-index配合定位元素使用才奏效

  7. line-height 不可以替代 vertical-align:middle

    1. line-height是一行的高度
    2. 实例中是一个div中有两行文字，需要使用vertical-align:middle来垂直居中，如果使用了line-height：div高，每一行都是div的高度，会超出div

  8. css中属性先后顺序原则：display float position影响盒子大的属性写到开头
