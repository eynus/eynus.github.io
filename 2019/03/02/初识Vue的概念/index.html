初识Vue的概念

Vue template编译过程

template编译过程

1.在 $mount 的时候执行 compile ，最终生成 render 函数。compile 方法分三步：

  1. parse 函数解析 template， 转化成了树状结构的 AST。
  2. optimize 函数优化静态内容
  3. generate 函数创建 render 函数字符串

2.接下来 Vue 执行 new watcher，监听绑定的数据

当这个 render 函数字符串在第一次 mount、或者绑定的数据更新的时候，都会被调用，生成 Vnode，更新到真正的 DOM 上。

Vue nextTick的理解

原理：

Vue 在修改数据后，视图不会立刻更新，而是等同一事件循环中的所有数据变化完成之后，再统一进行视图更新。

应用场景：

需要在视图更新之后，基于新的视图进行操作。

Vue mixin混合

理解：将多个组件中的共同的方法和数据提取出来放入mixin中，到时候在各个组件引入即可。引入后会和组件内已有的方法和数据进行混合生成一个新的，重合的话还是组件优先。
