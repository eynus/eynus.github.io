二、结构在HTML的体现

你所写的这些结构就造成了className,不过是英文版（html类只认英文）

重要的模块命名最好语义化一点，item适合小细节或者边边角角

一般不要直接用标题名，看本质

HTML教程

1.title也是搜索引擎搜索的结果

2.h1只能有一个（一个页面中

3.div最大功能：容纳其他元素

4.href:hyper text reference

5.img:alt的作用-图挂了会显示这是啥

6.abbr:abbreviation 缩写

7.code(行内元素) prew(block标签)
