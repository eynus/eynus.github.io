3.栅格

简介：

Bootstrap 提供了一套响应式、移动设备优先的流式栅格系统，随着屏幕或视口（viewport）尺寸的增加，系统会自动分为最多12列。

结构：

—.contianer/.container-fluid

.row
  1. 左右有15px的负margin (可利用)
  2. row的应用场景：不希望有左右两边的padding，此时可用外套一个row抵消
  3. .container 有padding 可以和row的负margin抵消
.col-xx-x
  1. 是向上兼容的！！！(比如设置col-sm-3,大于等于sm的宽度时该布局占比都是3)
  2. 针对小屏幕，覆盖栅格类(col-xs-3覆盖col-md-2)
栅格参数
超小屏幕 手机 (<768px) 小屏幕 平板 (≥768px) 中等屏幕 桌面显示器 (≥992px) 大屏幕 大桌面显示器 (≥1200px)
栅格系统行为 总是水平排列 开始是堆叠在一起的，当大于这些阈值时将变为水平排列C ← ←

11.表格

.table —基础表格样式

  • .table-striped —条纹状表格：给tbody的行添加斑马纹 需要有thead tbody生效

    条纹状表格是依赖 :nth-child CSS 选择器实现的，而它不被IE8 支持。

  • .table-bordered —带边框的表格

  • .table-hover —鼠标悬停状态对body的每一行做出响应

  • .table-condensed —让表格更紧凑

表格中每一行可以设置的状态类：

  1. .active
  2. .success
  3. .info
  4. .danger
  5. .warning

响应式表格

.table包裹在.table-responsive内，屏幕<768px时会有滚动条，>768px时自动消失。
