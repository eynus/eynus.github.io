实践笔记-碎知识

1.CSS

1.基本属性

  1. input和img是行内元素，为什么还可以设置宽高？

    替换元素一般有内在尺寸，所以具有width和height，可以设定。例如你不指定img的width和height时，就按其内在尺寸显示，也就是图片被保存的时候的宽度和高度。

    对于表单元素，浏览器也有默认的样式，包括宽度和高度。

  2. text-alian:left/right/center/justify

  3. line-height:比例/px值

  4. text-decoration:underline / overline / line-through

  5. background-size:cover ——图片充满整个容器

  6. 字体

    1. font-family:
      1. serif 衬线字体 (古典)
      2. sans-serif 非衬线字体
      3. “微软雅黑” ,“Microsoft Yahei”;
    2. font-weight:默认normal / bold / 值100-900区间
    3. font-size:inherit继承母元素大小

2.浮动注意事项

  • 注意：
    • 浮动元素撞到父级元素的“内壁”或和其他浮动元素摞到一起时才会停止“移动”，如果父级元素的空间不够，它们就另起一行。
    • fixed或absolute元素不浮动。
    • inline-block或inline元素在浮动后会变成block元素。(user agent stylesheet)
    • 未指定宽度的block元素的宽度会根据其内容长度而动态变化。
    • 一组“正常”的元素中如果有一个元素浮动会发生什么事情？
      • 浮动元素脱离文档流，后面的元素依次上移占据该元素空位
