动态插入n个li:较优解

  1. for循环，每次create一个li节点，浏览器每次在创建时都经历重排和重绘，重排重绘消耗性能。
  2. 用一个字符串变量拼接完n个li之后，集体插入。只消耗一次重排重绘。
