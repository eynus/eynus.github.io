动态缩放布局问题

  • 百分比布局
  • flex布局
  • rem 动态设置html的font-size大小 (js),设置为设备屏幕的百分比大小
  • vw vh 把每个屏幕分成一百份。

border:1px像素问题

  • meta标签：viewport 中根据设备dpr不同进行initial-scale 缩放

    js实现：创建一个meta标签，如果dpr>=2,content插入 :...scale=0.5.....

      如果dpr>=3, ....initial-scale=0.3....

  • css标签：先放大再缩小

    • border:1px ;width:2倍,height:2倍; transform:scale(0.5,0.5)
