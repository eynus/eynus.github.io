3.细节完善

  1. float/position:absolute会隐性的把内联元素转化为行级块元素，

    span元素为行级元素，其display默认属性为：inline；
    当将span设置浮动float：left/right;或position:absolute;后就可以设置宽高，

    原因是：span设置float：left/right;或position:absolute;后内部会将display属性变成：inline-block行级块元素；

  2. *{transition:color 200ms;}

  3. a元素有上下padding,margin,只不过没有撑开

  4. 行内元素的上下margin、padding是存在的，只不过没有实际撑开

  5. vertical-align:center

    vertical-align起作用的前提是元素为inline水平元素或table-cell元素

    所以inline-block的元素虽然里面内容是block的，但整盒对外而言是inline，有效
