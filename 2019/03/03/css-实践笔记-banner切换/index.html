  2. float引发的问题 ，用inline-block就没问题（？？？）

    不是label的icon设置float引发的问题，是label设置inline-block了。是因为label设置为inline-block之后，有回车会显示为空格，父元素设置font-size:0;即可解决！

  3. img大小需要单独设置，不随父级变化

  4. inline-block沿着默认的基线对齐，浮动元素紧贴顶部对齐，对vertical-align:center无效
