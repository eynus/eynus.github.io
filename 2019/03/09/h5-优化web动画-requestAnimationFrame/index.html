h5-优化web动画-requestAnimationFrame

setInterval

setTimeOut

计时器做动画的问题

  • 当前窗口不在动画页面时，计时器仍在工作——易崩溃
  • 回调函数时间不准确，在线程队列等待执行
  • 设置动画频率过高，过度绘制——出现掉帧

requestAnimationFrame

  • 当前敞口不再动画页面时，停止工作
  • 浏览器刷新频率(上限)时执行 (一秒刷新60次),无需设置时间间隔
  • 告诉浏览器这里有一个动画，浏览器会自动优化
