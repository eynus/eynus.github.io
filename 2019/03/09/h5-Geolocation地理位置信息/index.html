h5-Geolocation地理位置信息

Geolocation地理位置信息

(实际看浏览器内部实现方式)以下仅指理论上

获取当前地理位置

属性(它们在coords对象下) 描述
accuracy 经纬度的精度
altitude:null 海拔
altitudeAccuracy:null 设备前进方向
latitude:34.0653347 纬度
longitude:-118.24389099999999 经度
speed:null 设备前进速度
timestamp 获取地理位置信息的时间
属性 描述
code 1.用户拒绝位置服务
2.获取不到位置信息
3.获取信息超时
message 错误信息字符串
属性 描述
enableHighAccuracy 是否要求高精度的地理位置信息(鸡肋)
timeout 超时限制
maximumAge 缓存有效时间，期间内返回该缓存时间，若为0则重新获取新地理位置信息

实践

1.自己的电脑需要打开定位权限(win10)(坑爹==)

2.获取定位信息有时候会失败是因为geolocation只是获取地理位置信息的API,具体实现是由浏览器底层来实现的，就像网页打开获取信息有时失败是正常的。

3.火狐和谷歌的服务器都在国外，所以获取服务器下的地理位置信息需要翻墙。

4.这一点上，IE挺好用的
